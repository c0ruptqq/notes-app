# Set Notation

- $\cap$ - Intersection
- $\cup$ - Union
- A' - Not A 
- $\oslash$ - Empty set also {}
- $\xi$ - Universal set
- n(..) - Number of elements in the set
- C - Subset

# Common Rules

- $P(A\cup B) = P(A) + P(B) - P(A \cap B)$
- $P(A \cup B) = P(A) + P(B)$ - If events are mutually exclusive
- $P(A \cap B) = P(A) * P(B)$ - If events are independent
